1. 递推公式（Fibonacci 多项式的 GF(2) 版本）：

   1. f(0, x) = 1
   2. f(1, x) = x
   4. f(n, x) = x · f(n-1, x) + f(n-2, x)
   5. r(n) = deg(gcd(f(n, x), f(n, 1 + x)))

2. 其中：

   1. f(n, x) ∈ 𝔽₂[x]    （系数域为 GF(2)）
   2. deg : 𝔽₂[x] \ {0} → ℕ    （多项式次数）
   3. gcd : 𝔽₂[x] × 𝔽₂[x] → 𝔽₂[x]    （多项式最大公因子）

3. 性质：

   1. r(n) ≡ 0 (mod 2)    （偶数）
   2. r(n) ≤ n    （上界）
   3. r(n) ≤ r((n+1)k − 1)   (n, k ∈ ℕ)    （按倍数拉长的单调性）
